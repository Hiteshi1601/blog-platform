# Blog Platform

## Overview

Blog Platform is a full-stack web application that allows users to create, edit, publish, and manage blog posts. It provides a seamless writing and reading experience with secure user authentication, responsive design, and content management features.

## Features

* User Authentication (Sign Up, Login, Logout)
* Create, Edit, and Delete Blog Posts
* Rich Text Content Management
* View and Browse Published Blogs
* Search and Filter Posts
* Responsive Design for Mobile and Desktop
* Secure Backend API
* User-Friendly Interface

## Installation

### Clone the Repository

```bash
git clone <repository-url>
cd blog-platform
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file and add:

```env
MONGODB_URI=your_database_url
JWT_SECRET=your_jwt_secret
PORT=5000
```

### Run the Application

```bash
npm run dev
```
